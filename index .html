隐私政策 (Privacy Policy)

更新日期：2026年6月25日

WWPlayer（以下简称“本应用”）非常重视用户隐私。本隐私政策旨在说明本应用如何处理和保护您的数据。

1. 数据收集与使用 (Data Collection and Use)

本应用不会主动收集、出售或上传您的个人数据至开发者服务器。 您在本应用中添加的服务器地址、账号信息、访问令牌、播放偏好、缓存数据和本地设置，主要保存在您的本地 Windows 设备上，用于实现媒体库浏览、播放、播放记录同步等功能。

当您使用 Emby 服务器、TMDB、弹幕服务、字幕服务或其他由您配置的外部服务时，本应用会根据功能需要向这些服务发送必要请求。相关数据处理行为由对应服务提供方负责。

(English) This application does not actively collect, sell, or upload your personal data to the developer's servers. Server configurations, account information, access tokens, playback preferences, cache data, and local settings are primarily stored on your local Windows device. When you use Emby servers, TMDB, danmaku services, subtitle services, or other external services configured by you, the application may send necessary requests to those services to provide its features.

2. 本地数据与缓存 (Local Data and Cache)

本应用可能会在本地保存应用配置、媒体元数据缓存、图片缓存、播放记录缓存和日志文件，以提升加载速度、改善播放体验和辅助问题排查。您可以在应用设置中清理部分缓存数据。

(English) The application may store settings, media metadata cache, image cache, playback history cache, and log files locally to improve loading speed, playback experience, and troubleshooting.

3. 账号与服务器信息 (Account and Server Information)

您添加的 Emby 服务器账号、令牌和服务器地址仅用于连接您指定的服务器。本应用不会将这些信息发送给开发者。请您自行确认所连接服务器的可信度和隐私政策。

(English) Emby server accounts, tokens, and server addresses added by you are used only to connect to the servers you specify. The application does not send this information to the developer.

4. 购买与授权 (Purchases and Licensing)

如果本应用通过 Microsoft Store 提供付费解锁功能，购买、支付和授权状态由 Microsoft Store 处理。本应用可能会读取授权状态以判断功能是否已解锁，但不会直接处理您的支付信息。

(English) If paid unlock features are provided through Microsoft Store, purchases, payments, and license status are handled by Microsoft Store. The application may read license status to determine whether features are unlocked, but it does not directly process your payment information.

5. 权限声明 (Permissions Required)

本应用仅申请实现核心功能所必需的系统权限，例如本地文件读写、网络访问、媒体播放和缓存管理。我们不会利用这些权限窥探您的隐私。

(English) The application requests only the system permissions necessary for its core features, such as local file access, network access, media playback, and cache management.

6. 日志 (Logs)

本应用可能会在本地生成日志文件，用于排查播放失败、网络错误或应用异常。日志可能包含服务器地址、接口状态或错误信息。本应用不会自动上传日志；如果您主动向开发者提供日志，请先确认其中不包含敏感信息。

(English) The application may generate local log files for troubleshooting playback failures, network errors, or application issues. Logs are not uploaded automatically.
